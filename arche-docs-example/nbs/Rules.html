Rules¶

This notebook contains rules used in the library with examples. They are executed in report_all() with default parameters, unless stated otherwise.

Some definitions here are used interchangeably: * df - a dataframe which holds input data (from a job, collection or other source) * Scrapy cloud item - a row in a df * Items fields - columns in a df
