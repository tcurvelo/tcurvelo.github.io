Settings¶

Bitbucket access¶

To access Bitbucket, you have to set environment variables. You have these options:

  • Set up a username and password as BITBUCKET_USER and BITBUCKET_PASSWORD;

  • Set up BITBUCKET_TOKEN (see below).
